# Projeto vetclinic

Este é um programa básico para clínicas veterinárias para web.

O principais itens criados são:<br/>
  1. **Cadastro**<br/>
       - Tutores
       - Paciente
       - Funcionários
       - Vacinas
  2. **Agenda**
  3. **Atendimentos**
  4. **Criação de senha**
  5. **Validação dos campos**
  
  Adicionado um página como teste de estudo.
